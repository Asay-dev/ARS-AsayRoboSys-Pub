| **参数**                                                                                        | **价值**                       |
| --------------------------------------------------------------------------------------------- | ---------------------------- |
| **操作系统**                                                                                      |                              |
| 窗户                                                                                            | Windows 10 64 位或更高版本 Android |
| MacOS操作系统                                                                                     | 10.20 或更高版本                  |
| Ubuntu的                                                                                       | 最新的 LTS （20.04） 或更高版本        |
